Nuestro trabajo en
Software
Abril - Septiembre / 2025
Desarrollo Web 🌐
Como primera instancia, se llevo a cabo el desarrollo de una página web para la presentación del proyecto, donde se muestran los avances, desarrollo y funcionalidades del mismo. En esta etapa, se implementaron diversas herramientas y tecnologías para optimizar el proceso de desarrollo.
Junio - Octubre / 2025
Creación de Entorno Virtual en Raspberry Pi Zero 2W 💻
Se desarrolló un entorno virtual en la Raspberry Pi Zero 2W, permitiendo la ejecución de aplicaciones y servicios de manera eficiente. Este entorno virtualizado facilita la gestión de recursos y mejora el rendimiento del dispositivo.
Junio - Octubre / 2025
Implementación de BLE (Bluetooth Low Energy) 🛜
Para el desarrollo de la funcionalidad BLE, se utilizaron dos ESP32 C3 Super Mini. Para trabajar de una manera eficiente se aplico ESP-IDF que cuenta con un sistema de servidor y cliente BLE.
Julio - Agosto / 2025
Desarrollo de Placa en KiCad 🛠️
Se utilizó KiCad para el diseño de la placa personalizada, permitiendo la integración de los componentes necesarios para el funcionamiento del dispositivo. Este proceso incluyó la creación de esquemas y la disposición de los componentes en la placa. Se buscó un diseño lo más compacto y pequeño posible para ser comodo en la mano
